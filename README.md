## About The Project
This project brings the classic Tic-Tac-Toe game to life with a modern, interactive graphical user interface built using KivyMD. It offers a clean and intuitive experience for two players to enjoy the strategic fun directly on their desktop. Designed for simplicity and engagement, it showcases the capabilities of KivyMD for creating cross-platform applications with Python.

### Key Features
-   **Interactive GUI:** A visually appealing and responsive game board powered by KivyMD.
-   **Two-Player Local Gameplay:** Challenge a friend in a traditional head-to-head match on the same device.
-   **Clear Game State:** Intuitive display of whose turn it is and clear winning conditions.
-   **Simple Reset:** Easily start a new game after a win or draw.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites
Ensure you have Python installed on your system.
*   Python 3.x
*   `pip` (Python package installer)

### Installation
1.  Clone the repository:
    ```sh
    git clone https://github.com/krishujeniya/TIC-TOC-TUE-WITH-KIVYMD.git
    ```
2.  Navigate into the project directory:
    ```sh
    cd TIC-TOC-TUE-WITH-KIVYMD
    ```
3.  Install the required Python packages (Kivy and KivyMD):
    ```sh
    pip install kivy kivymd
    ```
4.  Run the game:
    ```sh
    python main.py
    ```
